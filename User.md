# User Dashboard - Full Specification

## Overview

Личный кабинет пользователя предоставляет доступ к управлению арендами, покупками, закладками, прогрессом чтения, штрафами и настройками уведомлений.

---

## 1. Разделы личного кабинета

### 1.1 Профиль

| Элемент | Описание |
|---------|----------|
| Личные данные | Имя, фамилия, email, телефон |
| Редактирование | Обновление профиля (с подтверждением пароля) |
| Смена пароля | Текущий пароль + новый пароль |
| Статус аккаунта | Активен / Заблокирован / Анонимизирован |
| Дата регистрации | Дата создания аккаунта |

### 1.2 Мои книги

#### Активные аренды
- Список текущих аренд с информацией:
  - Название книги, обложка
  - Тип аренды (Digital / Physical)
  - Дата начала, дата окончания
  - Оставшиеся дни
  - Статус (Active / Overdue)
  - Кнопка «Читать» (для Digital)
  - Кнопка «Инструкция по возврату» (для Physical)

#### История покупок
- Список купленных книг:
  - Название книги, обложка
  - Тип покупки (Physical / Digital)
  - Дата покупки
  - Цена
  - Статус (Completed / Cancelled)
  - Кнопка «Читать» (для Digital)
  - Кнопка «Скачать PDF» (для Digital, 1 раз)

#### История аренд
- Все завершённые аренды:
  - Название книги
  - Период аренды
  - Статус (Returned / Expired / Lost)
  - Были ли штрафы

### 1.3 История чтений

- Все прочитанные книги с прогрессом
- Для каждой книги:
  - Название, обложка
  - Последняя страница (`last_page`)
  - Всего страниц (`total_pages`)
  - Прогресс в процентах
  - Дата последнего чтения (`last_read_at`)
  - Кнопка «Продолжить чтение»

### 1.4 Закладки

- Список закладок по всем книгам:
  - Название книги
  - Номер страницы
  - Заметка к закладке (если есть)
  - Дата создания
  - Кнопка «Перейти к закладке» (открывает ридер на нужной странице)
- Фильтрация по книге
- Удаление закладок

### 1.5 Штрафы и задолженность

#### Текущие штрафы
- Список активных штрафов:
  - Название книги (из аренды)
  - Сумма штрафа (в тенге)
  - Статус (Pending / Paid / Waived)
  - Дата начисления
  - Причина (дни просрочки)

#### Баланс задолженности
- Отображение `total_debt` (общая сумма)
- Кнопка «Погасить долг» (CARD) → публикует `DebtPaymentRequested` в RabbitMQ
- Инструкция для оплаты наличными в библиотеке
- История оплат штрафов

#### Блокировка функций
При `total_debt > 0`:
- Нельзя арендовать новые книги
- Нельзя скачивать PDF
- Нельзя покупать книги (опционально)
- Виден баннер: «У вас есть задолженность. Погасите для разблокировки.»

### 1.6 Wishlist (Список желаний)

- Список книг, добавленных в wishlist
- Для каждой книги:
  - Название, обложка
  - Текущий статус доступности
  - Дата добавления
  - Статус уведомления (`notification_sent`)
  - Кнопка «Удалить из wishlist»
  - Кнопка «Арендовать» / «Купить» (если стала доступна)

### 1.7 Бронирования

#### Активные бронирования
- Список бронирований со статусом:
  - WAITING — в очереди
  - READY — готова к выдаче (с таймером 24 часа)
  - EXPIRED — истекло время
- Для каждого:
  - Название книги
  - Дата бронирования
  - Дата готовности (если READY)
  - Срок действия (если READY)
  - Кнопка «Забрать книгу» (если READY)

#### История бронирований
- Все завершённые/отменённые бронирования

### 1.8 Настройки уведомлений

Управление через Notification Service. Пользователь выбирает:

#### Каналы уведомлений
- Email (вкл/выкл)
- SMS (вкл/выкл)
- Push (вкл/выкл)

#### Типы уведомлений
- Напоминания о возврате (вкл/выкл)
- Уведомления о штрафах (вкл/выкл)
- Уведомления о доступности книг из wishlist (вкл/выкл)
- Уведомления о бронировании (вкл/выкл)

### 1.9 Отзывы

- Мои оставленные отзывы:
  - Название книги
  - Оценка (1-5 звёзд)
  - Текст рецензии
  - Статус (Pending / Approved / Rejected)
  - Дата создания
- Возможность редактировать отзыв (если Pending)
- Возможность удалить отзыв

## 2. Баланс задолженности — детальный флоу

### 2.1 Отображение

```
┌─────────────────────────────────────┐
│  Задолженность: 1 500 ₸             │
│                                     │
│  [Погасить онлайн (CARD)]           │
│  [Оплатить наличными в библиотеке]  │
│                                     │
│  ⚠️ Ваш аккаунт заблокирован        │
│  для новых аренд и скачивания PDF   │
└─────────────────────────────────────┘
```

### 2.2 Погашение онлайн (CARD)

1. Пользователь нажимает «Погасить долг»
2. Book Service публикует: `DebtPaymentRequested {user_id, amount}`
3. Payment Service слушает → создаёт invoice → списывает сумму → публикует: `DebtPaymentCompleted {user_id, amount}`
4. Book Service слушает → обновляет `user_debt.total_debt -= amount`
5. Если `total_debt = 0` → `is_blocked = false`
6. Notification Service → отправляет email об успешной оплате

### 2.3 Оплата наличными

1. Пользователь видит инструкцию: «Придите в библиотеку и оплатите штраф наличными»
2. Админ подтверждает оплату в панели
3. Book Service обновляет `user_debt.total_debt -= amount`
4. Если `total_debt = 0` → блокировка снимается

## 3. Онлайн-ридер (встроенный)

### 3.1 Функционал

- Отображение PDF в браузере
- Переход на последнюю прочитанную страницу
- Навигация по страницам
- Создание закладок (кнопка на каждой странице)
- Полноэкранный режим
- Масштабирование

### 3.2 Heartbeat

- Фронтенд отправляет heartbeat каждые 5 минут
- Payload: `{user_id, book_id, rental_id, page_number}`
- Если heartbeat не получен 15 минут → сессия закрывается
- Heartbeat обновляет:
  - `reading_sessions.last_heartbeat`
  - `reading_sessions.last_page`
  - `reading_progress.last_page`
  - `reading_progress.last_read_at`

## 4. События от User Dashboard

| Событие | Описание |
|---------|----------|
| `DebtPaymentRequested` | Запрос на оплату задолженности |
| `ReviewSubmitted` | Новый отзыв на модерацию |
| `WishlistItemAdded` | Книга добавлена в wishlist |
| `WishlistItemRemoved` | Книга удалена из wishlist |
| `ReservationCreated` | Создание бронирования |
| `ReservationCancelled` | Отмена бронирования |
| `BookmarkCreated` | Создание закладки |
| `BookmarkDeleted` | Удаление закладки |
| `NotificationSettingsUpdated` | Обновление настроек уведомлений |
